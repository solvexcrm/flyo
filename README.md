# Flyo ✈️

**All-in-one travel booking platform**

Find and book restaurants, hotels, entertainment and tours worldwide in one app.

## 🚀 Features

- 🌍 **Global Coverage** - Explore cities worldwide
- ⚡ **Instant Booking** - Book everything in seconds  
- 💰 **Best Prices** - Exclusive deals and discounts
- 🎯 **One Platform** - Restaurants, hotels, tours, entertainment
- 📱 **Beautiful UI** - Modern design with smooth onboarding

## 🛠️ Tech Stack

- **React Native** + **Expo**
- **TypeScript** (coming soon)
- **React Navigation** (coming soon)
- **Modern UI/UX** design

## 📱 Getting Started

### Prerequisites
- Node.js 16+
- Expo CLI: `npm install -g expo-cli`
- Expo Go app on your phone

### Installation

```bash
# Clone repository
git clone https://github.com/solvexcrm/flyo.git
cd flyo

# Install dependencies
npm install

# Start development server
expo start
```

### 📋 Development

```bash
# iOS Simulator
expo start --ios

# Android Emulator  
expo start --android

# Web Browser
expo start --web
```
